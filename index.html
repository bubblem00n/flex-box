Свойства FLEX-контейнера

1

display: flex;

display: -webkit-flex;

Назначение: превращает дочерные элементы в flex-элементы.

Возможные значения: flex / inline-flex.

flex

inline-flex

2

flex-direction: row;

-webkit-flex-direction: row;

Назначение: меняет расположение элементов по осям(располагаются в ряд или колону).

Возможные значения: row / row-reverse / column / column-reverse.

row

row-reverse

column

column-reverse

3

flex-wrap: nowrap;

-webkit-flex-wrap: nowrap;

Назначение: растположение элементов в одну или несколько строк.

Возможные значения: nowrap / wrap / wrap-reverse.

nowrap

wrap

wrap-reverse

4

flex-flow: row nowrap;

-webkit-flex-flow: row nowrap;

Назначение: запись, которая сокращает написание свойств flex-direction и flex-wrap.

Возможные значения: см свойства 2 и 3.

5

justify-content: flex-start;

-webkit-justify-content: flex-start;

Назначение: выравнивает flex-элементы по горизонтали (в случае с flex-direction: column работает вместо align-items т. к. направление оси поменялось).

Возможные значения: flex-start / flex-end / center / space-between / space-around.

flex-start

flex-end

center

space-between

space-around

6

align-items: strech;

-webkit-align-items: strech;

Назначение: выравнивает flex-элементы по вертикали.

Возможные значения: strech / flex-strat / flex-end / center / baseline.

strech

flex-start

flex-end

center

baseline

7

align-content: strech;

-webkit-align-content: strech;

Назначение: выравнивает строк flex-элементов вдоль поперечной оси.

Возможные значения: strech / flex-strat / flex-end / center / space-between / space-around.

strech

flex-start

flex-end

space-around

center

space-between
